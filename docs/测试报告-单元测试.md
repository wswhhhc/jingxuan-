### 修复的测试问题

| 问题 | 修复方式 |
|------|---------|
| WorkServiceImpl 构造参数不匹配（6 个测试文件） | 补齐 `DeepSeekApiClient`、`WorkMapper` 等缺失依赖 mock |
| `shouldSubmitWork`: 缺少 previewUrl 导致测试失败 | 补充 previewUrl 和附件（zip+mp4）mock |
| `shouldThrowWhenNoAttachment`: 无用 mock 导致 UnnecessaryStubbing | 改为 `selectList` mock |
| `shouldThrowWhenMemberAlreadyInBatch`: LambdaQueryWrapper 序列化失败 | 生产代码改用 `QueryWrapper`，测试 stub 标记 `lenient` |
