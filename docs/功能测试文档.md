## 1. 功能模块概述

系统按角色划分为 4 个端 + 1 个认证模块，共 5 大功能域：

| 模块 | 角色 | 核心功能 |
|------|------|---------|
| 认证 | 所有 | 登录/登出/信息/改密 |
| 公开端 | 匿名访客 | 作品浏览/详情/点赞/评论/排行榜 |
| 学生端 | 学生 | 作品 CRUD/提交审核/排名查询/通知 |
| 教师端 | 教师 | 作品查看/评分/排行榜/通知 |
| 管理端 | 管理员 | 审核/发布/用户/角色/字典/奖项/端口/日志/公告/敏感规则 |

## 3. 功能测试准入/准出

### 3.1 准入条件

- 被测版本已通过编译，无编译错误
- 单元测试通过率 ≥ 90%
- 核心 API 集成测试通过率 100%
- 测试数据库已初始化，包含必要的测试数据

### 3.2 准出条件

- 所有 P0 功能测试用例通过率 100%
- P1 功能通过率 ≥ 95%，未通过的已评估风险并记录
- 无 P0 级别缺陷遗留
- P1 缺陷已修复或已确认非阻断
- 功能测试报告已完成
