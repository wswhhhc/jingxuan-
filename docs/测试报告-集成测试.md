## 3. 端到端流程测试（9 项）

### 运行方式

```bash
mvn test -Dtest="EndToEndFlowTest"
```

### 测试结果

```

```

### 流程明细

| 编号 | 流程路径 | 跨角色 | 关键验证点 |
|------|---------|--------|-----------|
| E2E-1 | 创建作品 → 提交审核 → 管理员通过 → 发布 → 前台可见 | 学生→管理员→游客 | 状态流转正确、前台展示 |
| E2E-2 | 提交 → 驳回 → 修改 → 重新提交 → 通过 | 学生→管理员 | 驳回后状态重置、二次审核 |
| E2E-3 | 创建标签 → 前台按标签筛选 | 管理员→游客 | 标签持久化、公开查询 |
| E2E-4 | 评分提交 → 查询评分 → 查看历史 | 教师 | 评分 Upsert、历史记录 |
| E2E-5 | 未登录状态 → 点赞 → 取消 → 计数恢复 | 学生 | 点赞状态、计数正确性 |
| E2E-6 | 评论发表 → 列表+1 → 管理员删除 | 学生→管理员 | 评论 CRUD、权限隔离 |
| E2E-7 | 学生访问教师接口 → 403 | 学生 | Security 角色拦截 |
| E2E-8 | 教师访问管理接口 → 403 | 教师 | Security 角色拦截 |
| E2E-9 | 匿名访问需认证接口 → 401/403 | 游客 | Security 未认证拦截 |

## 4. 测试环境准备

```bash
# 建库
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS jingxuan_test CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 导入表结构 + 测试数据（需先移除 SQL 中的 USE jingxuan; 语句）
sed '/^USE jingxuan;/d; /^CREATE DATABASE/d' sql/base/init_schema.sql | mysql -u root -p jingxuan_test
sed '/^USE jingxuan;/d' sql/business/work_schema.sql | mysql -u root -p jingxuan_test
sed '/^USE jingxuan;/d' sql/business/test_data.sql | mysql -u root -p jingxuan_test

# 增量迁移
for f in sql/business/2026-*.sql; do sed '/^USE jingxuan;/d' "$f" | mysql -u root -p jingxuan_test; done
```

测试数据库连接配置在 `application-test.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/jingxuan_test?useUnicode=true&characterEncoding=UTF-8&serverTimezone=Asia/Shanghai
    username: root
    password: ${DB_PASSWORD}
```
