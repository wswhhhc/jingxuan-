## 二、项目背景

某 IT 学院每年产生大量学生课程作品、实训项目与毕业设计成果。传统的作品提交、展示、评分和汇总依赖人工收集与线下管理，存在资料分散、统计困难、展示效果弱、评审效率低等问题。

为解决上述痛点，我们开发了**菁选校园作品展示平台**，实现从作品提交 → 审核 → 评分 → 排行 → 发布展示的一体化管理。

**关键创新点：**
- 引入「审核」环节（原始需求中学生提交后直接展示，我们增加了管理员审核），确保展示内容合规
- 审核状态与发布状态分离，管理员可在审核通过后灵活控制展示时机
- 四维度匿名评分（创新性25 + 技术难度25 + 完成度30 + 实用性20），避免人情分

### 7.2 评分 Upsert 原子操作

教师评分支持修改，使用 MySQL INSERT ... ON DUPLICATE KEY UPDATE 实现：

```java
@Insert("""
    INSERT INTO work_score (id, work_id, teacher_id, innovation, difficulty, completion, practicality, total, comment, batch_id)
    VALUES (#{id}, #{workId}, #{teacherId}, #{innovation}, #{difficulty}, #{completion}, #{practicality}, #{total}, #{comment}, #{batchId})
    ON DUPLICATE KEY UPDATE
    innovation = VALUES(innovation), difficulty = VALUES(difficulty),
    completion = VALUES(completion), practicality = VALUES(practicality),
    total = VALUES(total), comment = VALUES(comment)
""")
void upsert(WorkScore workScore);
```

### 7.3 双层内容安全审核（DFA + AI）

```java
public SensitiveCheckResult check(String text) {
    // 第一层：DFA 敏感词高速过滤（Trie 树 O(n) 匹配）
    if (dfaMatcher.containsSensitiveWord(text)) {
        return SensitiveCheckResult.rejected("内容包含违禁词");
    }
    // 第二层：DeepSeek AI 语义审核
    SensitiveCheckResult aiResult = deepSeekApiClient.checkText(text);
    // AI 不可用时降级：返回警告而非拒绝
    return aiResult != null ? aiResult : SensitiveCheckResult.warning("AI审核不可用，已转入人工复核");
}
```

### 7.4 文件上传双重校验（扩展名 + 魔数）

```java
// 第一关：扩展名白名单
if (!ALLOWED_EXTENSIONS.contains(extension)) {
    throw new BizException("不支持的文件类型");
}
// 第二关：Hutool 文件魔数检测（防 XSS）
if (isRealTypeMismatch(file, extension)) {
    throw new BizException("文件内容与扩展名不匹配，疑似风险文件");
}
```

### 7.5 统一响应 + 全局异常处理

```java
// 统一返回格式
@Data
public class Result<T> {
    private int code;      // 0=成功
    private String msg;    // 提示信息
    private T data;        // 数据
}

// 全局异常处理
@RestControllerAdvice
public class GlobalExceptionHandler {
    @ExceptionHandler(BizException.class)
    public Result<?> handleBizException(BizException e) {
        return Result.fail(e.getMessage());
    }
}
```
