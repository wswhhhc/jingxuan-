## 1. 系统简介

菁选校园作品展示平台是面向 IT 学院的学生作品集中管理、展示、评分与排行系统。学生可以提交课程作品和毕业设计，教师匿名评分，管理员审核发布，游客浏览展示。

### 角色说明

| 角色 | 功能 |
|------|------|
| 学生 | 提交作品、查看审核结果、查看评分与排名 |
| 教师 | 匿名评分、查看排行榜 |
| 管理员 | 审核作品、发布管理、用户管理、系统配置 |
| 游客 | 浏览作品、查看排行榜 |

## 3. 学生端操作

### 3.1 学生首页

登录后进入学生首页，页面显示：
- 个人作品概览（作品数、审核状态）
- 快捷操作入口（提交作品、我的作品）

### 3.2 提交作品

1. 点击「提交作品」进入作品编辑页面
2. 填写作品信息：

   | 字段 | 说明 | 必填 |
   |------|------|------|
   | 作品名称 | 作品标题，不超过 200 字 | ✅ |
   | 作品简介 | 作品功能与特点描述 | |
   | 技术栈 | 使用的技术，如 `Java/Spring Boot/Vue3` | |
   | 指导教师 | 指导老师姓名 | |
   | 服务器访问地址 | 作品部署后的在线访问链接 | |
   | 运行说明 | 如何运行该作品 | |

3. **添加团队成员：**
   - 点击「添加成员」
   - 输入成员姓名、学号、班级
   - 队长排在第一位（`isLeader=1`）

4. **上传附件：**
   - 点击「上传文件」选择 zip/rar/7z 格式的源代码压缩包
   - 点击「上传视频」选择 mp4 格式的演示视频
   - 上传成功后文件会显示在列表中

5. **保存与提交：**
   - 点击「保存草稿」— 暂存作品，后续可继续编辑
   - 点击「提交审核」— 提交给管理员审核，提交后不可再编辑

### 3.3 我的作品

进入「我的作品」页面，查看所有已提交的作品列表：

| 状态 | 说明 | 可操作 |
|------|------|--------|
| 草稿 | 未提交，可继续编辑 | 编辑 / 删除 / 提交 |
| 已提交 | 等待管理员审核 | 不可操作 |
| 已驳回 | 管理员驳回，可修改后重新提交 | 编辑 / 重新提交 |
| 已通过 | 审核通过 | 查看详情 |

- 点击作品名称可查看详情
- 被驳回的作品会显示驳回原因

### 3.4 我的评分与排名

1. 进入「我的评分」页面
2. 查看自己在各评分批次中的得分：
   - 各维度均分（创新性/技术难度/完成度/实用性）
   - 综合平均分
   - 当前排名
   - 评分数（几位教师评分）

### 3.5 消息通知

1. 页面右上角的铃铛图标显示未读通知数量
2. 点击进入通知列表：
   - 查看通知详情
   - 点击「标记已读」
   - 点击「全部已读」

## 4. 教师端操作

### 4.1 教师工作台

登录后进入教师工作台，显示：
- 待评分作品数量
- 已完成评分数量
- 评分完成率
- 活跃批次信息

### 4.2 作品评分

1. 进入「作品评分」页面
2. 系统随机展示已通过且未评分的作品（匿名，不显示学生姓名）
3. 点击作品查看详情
4. 在评分表单中输入四维度分数：

   | 维度 | 满分 | 说明 |
   |------|------|------|
   | 创新性 | 25 分 | 作品创意与创新程度 |
   | 技术难度 | 25 分 | 技术实现复杂度 |
   | 完成度 | 30 分 | 功能完整性与代码质量 |
   | 实用性 | 20 分 | 实际应用价值 |

5. 填写评语（可选）
6. 点击「提交评分」
7. 如需修改，重新评分后再次提交即可覆盖

### 4.3 评分历史

进入「评分历史」页面：
- 查看自己所有历史评分记录
- 显示作品名称、各维度分数、总分、评分时间

### 4.4 排行榜

进入「排行榜」页面：
- 查看综合排行榜，按总分降序排列
- 支持按评分批次切换
- 支持按技术栈分类筛选
- 点击「刷新缓存」可获取最新排名数据

## 5. 管理端操作

### 5.1 控制台

登录后进入管理控制台，页面显示：
- 平台统计数据（作品总数、用户数、评分批次数等）
- 评分分布图表
- 最近提交作品动态

### 5.2 审核管理

**审核列表：**
1. 进入「审核管理」页面
2. 作品列表展示所有已提交的作品
3. 支持按状态、班级、关键词、提交时间范围筛选

**审核操作：**
1. 点击作品查看完整详情
2. 点击「审核通过」— 作品状态变为「已通过」后可发布
3. 点击「驳回」— 填写驳回原因后提交，学生端可见

**发布管理：**
1. 已通过作品可进行以下操作：
   - **发布** — 作品在前台公开展示
   - **下线** — 作品从前台移除
   - **设为精选** — 设置精选标记并填写在线体验地址

### 5.3 评审批次管理

1. 进入「评审批次管理」页面
2. **创建批次：** 填写批次名称、起止时间、班级范围
3. **管理批次：** 可启用/停用、编辑批次信息
4. **排行榜公示：** 批次评分结束后点击「公示排行榜」，学生和前台可见排名
5. **批次通知：** 可编辑并发布批次通知，范围内的学生将收到通知

### 5.4 用户管理

1. 进入「用户管理」页面
2. **创建用户：** 填写用户名、姓名、角色、班级等信息
3. **编辑用户：** 修改用户信息
4. **状态管理：** 启用/禁用用户账号
5. **批量导入：** 批量导入用户
6. **AI 导入：** 粘贴文本由 AI 自动解析并导入

### 5.5 角色权限

1. 进入「角色权限」页面
2. 查看角色列表
3. 编辑角色信息
4. 为角色分配菜单权限（勾选菜单树）

### 5.6 奖品配置

1. 进入「奖品配置」页面
2. 配置奖项（等级、名称、奖品说明、名额）
3. 按评分排名发放奖品
4. 查看发放记录，可取消发放

### 5.7 其他管理功能

| 功能 | 入口 | 说明 |
|------|------|------|
| 公告管理 | 公告管理页 | 创建/编辑/发布公告 |
| 数据字典 | 字典管理页 | 维护班级、技术栈等基础数据 |
| 内容审核规则 | 审核规则页 | 配置 DeepSeek API 规则、连通性测试 |
| 评论管理 | 评论管理页 | 查看/删除违规评论 |
| 操作日志 | 日志页 | 查询关键操作日志 |
| 消息通知 | 通知页 | 查看管理员通知 |

## 6. 前台公开浏览

### 6.1 作品列表

- 访问首页即可浏览已发布的作品
- 支持按技术栈、班级、提交时间筛选
- 分页浏览

### 6.2 作品详情

点击作品进入详情页：
- 查看作品完整介绍
- 团队成员信息
- 附件下载
- 视频演示播放
- 在线体验链接跳转

### 6.3 评论互动

- 登录用户可在作品下方发表评论
- 可回复已有评论
- 支持点赞/取消点赞

### 6.4 公共排行榜

- 访问排行榜页面查看综合排名
- 可按评分批次切换
- 可按技术栈分类查看
- 获奖作品标注等级（一等奖/二等奖/三等奖）

## 7. 常见问题

| 问题 | 解决方法 |
|------|---------|
| 忘记密码 | 联系管理员重置密码 |
| 上传文件失败 | 检查文件格式（仅支持 zip/rar/7z/jpg/png/gif/mp4），单文件不超过 200MB |
| 作品无法提交 | 检查是否已上传源代码压缩包和演示视频 |
| 看不到排行榜 | 联系管理员确认排行榜是否已公示 |
| 页面显示异常 | 尝试刷新页面或清除浏览器缓存 |
